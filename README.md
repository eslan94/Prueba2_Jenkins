# Repositorio Prueba2_Jenkins
# Prueba 2 - Automatización de Acciones Avanzadas

Este repositorio contiene la solución del Caso Práctico 3 de automatización con Selenium y Python.

## Requerimientos
* Implementación de la clase `Actions`.
* Método genérico para `Drag and Drop`.
* Validaciones de resultados esperados.

## Cómo ejecutarlo localmente
1. Clonar el repositorio: `git clone https://github.com/Eslan94/Prueba-2.git`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar el test: `pytest -s test_actions.py`
